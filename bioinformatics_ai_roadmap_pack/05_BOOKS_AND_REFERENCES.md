# Books and References

**Updated:** 2026-06-08

## Free / Open First

| Resource | Best For | Link | Notes |
|---|---|---|---|
| Computational Genomics with R | genomics data analysis with R | https://compgenomr.github.io/book/ | Excellent bridge from programming/statistics to genomics. |
| Bioconductor Books | RNA-seq, single-cell, spatial, Bioconductor workflows | https://bioconductor.org/books/release/ | Use release version for current package context. |
| Orchestrating Single-cell Analysis with Bioconductor | single-cell foundations | https://bioconductor.org/books/release/OSCA/ | Deep, rigorous single-cell workflow material. |
| Bioconductor RNA-seq Workflow | RNA-seq differential expression | https://www.bioconductor.org/packages/release/workflows/vignettes/rnaseqGene/inst/doc/rnaseqGene.html | Practical workflow. |
| Bioinformatics Algorithms | algorithms and sequence analysis | https://www.bioinformaticsalgorithms.org/ | Free chapters/videos; full book optional. |
| NCBI Bookshelf / NCBI training | databases and tools | https://www.ncbi.nlm.nih.gov/home/learn/ | Official source. |
| Data Carpentry Genomics | command line and genomics workflows | https://datacarpentry.github.io/genomics-workshop/ | Very practical. |
| Galaxy Training Network | hands-on workflows | https://training.galaxyproject.org/ | Good when CLI setup is painful. |

## Paid / Optional but Valuable

Do not buy these immediately. Use free resources first. Buy later if you are committed.

| Book | Best For | Notes |
|---|---|---|
| Bioinformatics Data Skills — Vince Buffalo | command line, data hygiene, reproducible bioinformatics | Very aligned with practical bioinformatics work. |
| Bioinformatics Algorithms — Compeau & Pevzner | algorithms and computational biology | Great if you like algorithmic depth. |
| Bioinformatics and Functional Genomics — Pevsner | broad biology + bioinformatics reference | More biology-heavy and comprehensive. |
| Biological Sequence Analysis — Durbin et al. | classic probabilistic sequence modeling | Older but deep for HMMs/alignment. |
| Introduction to Statistical Learning / ESL | ML/statistical foundations | You already know ML, but useful for rigorous stats. |

## What to Read When

### First month

- [ ] Khan Academy central dogma.
- [ ] NCBI basics.
- [ ] Data Carpentry Genomics setup and command line.
- [ ] First Rosalind problems.

### Months 2–3

- [ ] Bioinformatics Algorithms selected chapters.
- [ ] Computational Genomics with R selected chapters.
- [ ] Bioconductor Intro.
- [ ] RNA-seq workflow.

### Months 4–6

- [ ] DESeq2 paper.
- [ ] edgeR paper.
- [ ] Snakemake tutorial/docs.
- [ ] Nextflow/nf-core docs.

### Months 6–9

- [ ] OSCA / Scanpy docs.
- [ ] AlphaFold 2/3 papers.
- [ ] scGPT paper.
- [ ] Nucleotide Transformer / Evo papers.

### Months 9+

- [ ] Foundation models in bioinformatics survey.
- [ ] genome language model survey.
- [ ] single-cell foundation models review.
- [ ] agentic workflow papers/tools.

## Notes on Piracy and Access

Some search results may surface PDFs of paid books from unofficial mirrors. Do not rely on piracy. Prefer official free chapters, institutionally available copies, library access, or open alternatives.

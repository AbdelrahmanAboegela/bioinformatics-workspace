# Resource Catalog: English + Arabic

**Updated:** 2026-06-08

This catalog prioritizes free/open resources. Some platforms offer free audit but charge for certificates. Use certificates only if you need them; the learning is the important part.

## Resource Selection Rules

- Prefer official documentation and institutional training.
- Use Arabic resources as intuition/support, but verify commands and tool details from official English docs.
- Prefer resources with hands-on exercises.
- Prefer resources that teach reproducible workflows, not just screenshots.
- Avoid relying on old unmaintained tutorials for exact tool commands.

### How to use Arabic resources effectively

- [ ] Watch Arabic videos before/after English lectures to reduce friction.
- [ ] Do not copy old installation commands blindly.
- [ ] Verify package versions from official docs.
- [ ] Write your own Arabic/English glossary.

## Recommended Learning Path Using Resources

### Biology foundations

- [ ] Khan Academy Central Dogma.
- [ ] Arabic intro videos for biological intuition.
- [ ] NCBI basic tutorials.

### Bioinformatics algorithms

- [ ] Bioinformatics Algorithms free chapters/videos.
- [ ] Rosalind Python Village and Bioinformatics Stronghold.
- [ ] MIT OCW 7.91J selected lectures.

### Practical genomics

- [ ] Data Carpentry Genomics Workshop.
- [ ] Galaxy Training Network for visual workflow intuition.
- [ ] NCBI training for BLAST/GEO/SRA.

### RNA-seq and expression

- [ ] HarvardX Biomedical Data Science / Genomics Data Analysis.
- [ ] Introduction to Bioconductor.
- [ ] Bioconductor RNA-seq workflow.
- [ ] Rockefeller RNA-seq with Bioconductor course: https://rockefelleruniversity.github.io/RU_RNAseq/

### Single-cell

- [ ] Bioconductor OSCA book.
- [ ] Scanpy tutorials: https://scanpy.readthedocs.io/en/stable/tutorials/index.html
- [ ] Seurat tutorials: https://satijalab.org/seurat/articles/get_started.html
- [ ] scvi-tools tutorials: https://docs.scvi-tools.org/en/stable/tutorials/index.html

### Proteins and structure

- [ ] UniProt tutorials/docs.
- [ ] AlphaFold DB training.
- [ ] PDB-101: https://pdb101.rcsb.org/
- [ ] EBI structural bioinformatics training.

### Reproducible workflows

- [ ] Snakemake tutorial.
- [ ] Nextflow training.
- [ ] nf-core docs and pipelines.
- [ ] Galaxy Training if you want workflow intuition before CLI workflows.

## Modality Mix

| Modality | Use it for | Examples |
|---|---|---|
| Video | intuition and first exposure | Khan Academy, Arabic playlists, MIT lectures |
| Textbook/book | depth and reference | Bioinformatics Algorithms, Computational Genomics with R, Bioconductor books |
| Problem platform | coding muscle | Rosalind |
| Tutorial notebooks | reproducible analysis | Bioconductor workflows, Scanpy tutorials |
| Official docs | commands/API accuracy | NCBI, Snakemake, Nextflow, nf-core, UniProt |
| Papers | research literacy | BLAST, DESeq2, AlphaFold, scGPT, Nucleotide Transformer |
| Projects | portfolio and real skill | see `03_PROJECT_PORTFOLIO.md` |
